# Advanced Excel Sales Analytics and Dashboard System

## Overview

This project demonstrates advanced data analysis and visualization techniques using Microsoft Excel. It focuses on extracting meaningful business insights from customer sales data through analytical tools, statistical methods, and interactive dashboards.

The project combines Excel formulas, pivot tables, regression analysis, and data visualization to support data-driven decision-making.

---

## Project Objective

The objective of this project is to:

- Perform advanced data analysis using Excel  
- Apply statistical and analytical techniques  
- Build interactive dashboards for insights  
- Use data storytelling to present findings effectively  

---

## Dataset Description

The dataset contains customer-level sales information with the following fields:

- Customer_ID  
- Customer_Name  
- Region  
- Product_Category  
- Sales  
- Quantity  
- Discount  
- Order_Date  
- Profit  

---

## Tools and Features Used

- Microsoft Excel  
- Conditional Formatting  
- WHAT-IF Analysis (Scenario Manager)  
- Data Analysis Toolpak  
- Pivot Tables and Charts  
- Advanced Excel Formulas  
- Dashboard Design  

---

## Tasks Performed

### Conditional Formatting
- Highlighted top 10 customers based on total purchase value  
- Enabled quick identification of high-value customers  

---

### WHAT-IF Analysis
- Used Scenario Manager to analyze profit under different discount levels  
- Created scenarios:
  - No Discount  
  - Moderate Discount  
  - High Discount  

---

### Regression Analysis
- Performed linear regression using Data Analysis Toolpak  
- Analyzed relationship between Sales and Profit  
- Interpreted R-square and coefficients  

---

### Descriptive Statistics
- Generated summary statistics:
  - Mean, median  
  - Standard deviation  
  - Minimum and maximum  

---

### Monthly Sales Growth
- Calculated monthly growth trends  
- Used conditional formatting with arrows for visualization  

---

### Timestamp Tracking
- Added timestamp using NOW() function  
- Tracked last update time  

---

### High-Value Customer Identification
- Used:
  - INDEX  
  - MATCH  
  - FILTER  
- Identified customers contributing maximum revenue  

---

### Pivot Table Analysis
- Total sales by region  
- Total sales by product category  
- Used pivot tables for summarization and visualization  

---

### Data Visualization

- Bar Chart for regional sales comparison  
- Line Chart for monthly sales trend  
- Pie Chart for product category contribution  

---

### Dashboard and Data Storytelling
- Designed interactive dashboard showing:
  - Total Sales  
  - Total Profit  
  - Top Customers  
  - Sales Trends  
- Presented insights using charts and KPIs  

---

## Key Insights

- A small group of customers contributes a major share of revenue  
- High discounts reduce profit beyond a certain threshold  
- Sales and profit have a positive correlation  
- Certain regions and product categories consistently perform better
